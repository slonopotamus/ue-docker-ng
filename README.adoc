= ue-docker-ng
:source-highlighter: rouge
:icons: font
:toc:

A modern, multi-stage Docker build pipeline for producing Unreal Engine editor images for Linux and Windows.

This project uses Docker Buildx Bake for composable, cache-efficient builds.

NOTE: This project is inspired by link:https://github.com/adamrehn/ue4-docker[adamrehn/ue4-docker].

== Status

Linux works.

Windows is blocked by https://github.com/moby/moby/issues/53030.

== Features

* **Multi-stage builds** - keep the final image lean by copying only the installed engine binaries.
* **Multi-platform** - produces images for both Linux and Windows.
* **Buildx Bake** - targets are defined in link:docker-bake.hcl[docker-bake.hcl], making builds reproducible and easy to customise.
* **Configurable source** - switch the Unreal Engine repository and tag via variables.
* **Compressed OCI images** - output images use zstd compression by default for smaller storage footprint.

== Usage

Build a Linux image::
+
[source,bash]
----
$ GIT_AUTH_TOKEN=<token> docker buildx bake https://github.com/slonopotamus/ue-docker-ng.git linux-minimal
----

Build a Windows image::
+
[source,bash]
----
$ GIT_AUTH_TOKEN=<token> docker buildx bake https://github.com/slonopotamus/ue-docker-ng.git windows-minimal
----

=== Variables

All variables are defined in link:docker-bake.hcl[docker-bake.hcl] and can be overridden via environment variables::

[source,bash]
----
$ source-tag=5.8.0-release docker buildx bake linux-minimal
----

[cols="2,1,2"]
|===
|Variable |Default |Description

|`source-repository`
|`https://github.com/EpicGames/UnrealEngine.git`
|Git URL for the Unreal Engine source

|`source-tag`
|`5.7.4-release`
|Git tag or branch to checkout

|`changelist`
|`auto`
|Override changelist number in `Build.version`; `auto` derives it from the tag

|`tag-namespace`
|`slonopotamus`
|Docker image tag prefix

|`image-outputs`
|`[{ type = "image", oci-mediatypes = true, compression = "zstd", push = true, unpack = false }]`
|Output configuration for the final images (type, mediatypes, compression, push, unpack)

|`linux-setup-args`
|`--exclude=Android --exclude=Mac --exclude=Win32 --exclude=Win64`
|Arguments passed to `Setup.sh` on Linux

|`windows-setup-args`
|`--exclude=Android --exclude=Mac --exclude=Linux`
|Arguments passed to `Setup.bat` on Windows

|`common-buildgraph-args`
|`[-set:HostPlatformOnly=true -set:WithClient=true -set:WithDDC=false -set:WithServer=true]`
|Arguments passed to BuildGraph on all platforms

|`linux-buildgraph-args`
|(empty)
|Extra flags appended to the Linux BuildGraph command (on top of `common-buildgraph-args`)

|`windows-buildgraph-args`
|(empty)
|Extra flags appended to the Windows BuildGraph command (on top of `common-buildgraph-args`)

|`linux-baseimage`
|`docker-image://ubuntu:22.04`
|Base image for Linux builds

|`windows-baseimage`
|`docker-image://mcr.microsoft.com/windows/server:ltsc2022`
|Base image for Windows builds

|`linux-platforms`
|`["linux/amd64"]`
|Target platforms for Linux builds

|`windows-platforms`
|`["windows/amd64"]`
|Target platforms for Windows builds
|===

== Project Structure

[source]
----
.
├── docker-bake.hcl                   # Buildx bake target definitions
├── linux/
│   ├── base/                         # OS prerequisites, user setup
│   │   └── install-prerequisites.sh  # Installation script
│   ├── source/                       # Clone repo, run Setup.sh to download dependencies
│   ├── builder/                      # Create Installed Build
│   │   └── set-changelist.py         # Changelist injection utility
│   └── minimal/                      # Final image
└── windows/
    ├── base/                         # OS prerequisites, Chocolatey, Python
    │   └── install-prerequisites.ps1 # Choco/PowerShell installation script
    ├── source-prep/                  # Source checkout
    ├── vs/                           # Visual Studio Build Tools
    │   └── install-vs.py             # VS installation script
    ├── source/                       # Run Setup.bat to download dependencies
    │   └── patch-setup-win.py        # Setup script patcher
    ├── builder/                      # Build UBT, create Installed Build
    │   └── set-changelist.py         # Changelist injection utility
    └── minimal/                      # Final image
----

== Requirements

* **Docker** 29 or later, with https://docs.docker.com/engine/storage/containerd/[containerd image storage] enabled
* **Git authentication token** - specify  `GIT_AUTH_TOKEN` environment variable to authenticate with the Unreal Engine repository
